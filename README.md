# 一、功能介绍
- 用户注册、登录、找回密码
- 文章管理
- 分类管理
- 标签管理
- 用户管理
- 评论管理
- 用户未登录根据文章访问量推荐
- 用户登录后根据用户喜好推荐
- 系统分用户和管理员2个角色

该博客是极为轻量级博客，代码十分清晰简单


# 二、技术组成
- 1、SpringBoot 或 SSM
- 2、MyBatis
- 3、Thymeleaf
- 4、Bootstrap
- 5、MySQL
